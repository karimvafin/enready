# EnReady - приложения для быстрого изучения и повторения английского языка по заданной теме

## Описание
Это приложение на IOS, которое автоматически генерирует материалы для изучения или повторения английского языка по заданной теме.

## Использование
При открытии приложения появляется интерфейс как у чат бота. То есть показывается иконка приложения и строка запроса, куда пользователь печатает свой запрос (в нем по умолчанию написано "What do you want to learn?" незаметным шрифтом). Также есть вкладка меню слева, где пользователь может выбрать уже введенные ранее запросы. После запроса появляется отдельная вкладка в меню по теме запроса (которая автоматически суммаризируется исходя из запроса пользователя). В этой вкладке должны быть следующие вкладки: карточки (как в quizlet), заучивание (как в quizlet), диалог (это мини-диалог на тему запроса с языковой моделью, где модель общается с пользователем и умеет задавать ему вопросы). Карточки и упражнения для заучивание генерируются автоматически с помощью LLM.

### Карточки
Пусть в начале будет фиксированное количество карточек - 20. То есть для каждой темы должно генерироваться 20 самых часто встречающихся слов / выражений с обьяснением и переводом на русский.

### Заучивание
Здесь должны идти различные упражнения от простых до сложных. Пусть это будут следующие упражнения:
1) Дается слово на английском из карточек в случайном порядке, и нужно из нескольких вариантов выбрать правильный перевод (и так для всех слов из карточек)
2) Дается набор предложений, в которых пропущены слова / выражения из карточек, и нужно вставить в предложение правильный вариант из набора (и так для всех слов из карточек)

## MVP
Чтобы протестировать эту идею, сначала нужно создать MVP. Поэтому не будем начинать с цельного приложения на IOS, сначала создадим web-приложение, которое выглядит как мобильное, но с тем же фукнционалом. Также в MVP не будет подключено API языковой модели, на любой запрос будет захардкожена одна тема для следующего запроса "help me to prepare for english interview". То есть для любого запроса просто будут появляться захардкоженные карточки (пусть будет 15 карточек) и упражнения для этой определенной темы.
